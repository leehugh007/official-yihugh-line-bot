## CREDENTIALS.md 已補的 Supabase 直連路徑

**背景**：Supabase MCP token 授權只有 coach-line-bot，官方/阿算共用的 project 不在 MCP。

**解法**：用 `official-yihugh-line-bot/.env.local` 的 `SUPABASE_URL` + `SUPABASE_KEY` 直連，pattern：
```js
import { createClient } from '@supabase/supabase-js';
import { readFileSync } from 'fs';
const env = Object.fromEntries(
  readFileSync('./.env.local', 'utf-8').split('\n')
    .filter(l => l && !l.startsWith('#') && l.includes('='))
    .map(l => { const [k, ...rest] = l.split('='); return [k.trim(), rest.join('=').trim().replace(/^["']|["']$/g, '')]; })
);
const supabase = createClient(env.SUPABASE_URL, env.SUPABASE_KEY, { auth: { persistSession: false } });
// SELECT / UPDATE
```

臨時 script 建議用**已在 session 跑過的檔名**（`_tmp_reset_yixiu.mjs` 這類），sandbox 對新檔名會 deny。

詳見：`CREDENTIALS.md` 新增的「Supabase 本地直連」小節。
